# 概要

シンプルなシューティングゲームです。クリア条件などは無く、ミスをするまでのスコアで競います。レベルが上がるほどに敵が増えていきます。

# 対応プラットフォーム

以下のプラットフォームに対応しています．

- Cygwin
- macOS
- Linux

# ビルドに必要なもの

このプログラムをビルドするには以下のプログラム・ライブラリが必要です．

- make
- gcc または clang
- OpenGL
- GLUT

# ビルド方法

ソースコード類が配置されているディレクト内でコマンドラインで以下のコマンドを実行します．

```shell
make
```

ビルドに成功するとディレクト内に`j19426`という実行ファイルが生成されます．

# 実行方法

生成された実行ファイル`j19426`をダブルクリックするか，コマンドラインで以下のコマンドを実行します．

```shell
./j19426
```

# 遊び方

使用するキーは以下の通りです．

- Zキー (決定・ショット)
- 矢印キー (メニュー選択・自機移動)
- Pキー (デバッグ用)

起動するとタイトル画面が表示されます．下のメニューは矢印キーで選択し，Zキーで決定できます．`START`はゲーム開始，`QUIT`はプログラムが終了します．

ゲーム画面では敵が画面上部からどんどん出現します．Zキーでショットを撃つことができるのでこれを敵に当てて倒しましょう．
敵は特定のパターンで弾を撃ちます．この弾に自機が当たるとゲームオーバーです．自機の当たり判定は中心の白い部分のみなのでうまく立ち回りましょう．なお，敵自体には当たり判定はありません．
敵を倒すと画面下部に表示されているスコアが増えます．スコアの増分は敵によって異なります．また，スコアが増えれば増えるほどレベルが上がり，敵の出現頻度が増加します．

ゲームオーバーになると`CONTINUE`と`TITLE`のメニューが出現します．`CONTINUE`を選ぶと始めからやり直せます．`TITLE`はタイトル画面に戻ります．

デバッグ用に`P`キーを押すとゲームが一時的に停止します．再び`P`キーを押すことで再開します．
